Herbruikbaarheid

Een gewenste eigenschap die objectoriëntatie populair heeft gemaakt, is herbruikbaarheid van delen van het systeem. Soms kan het veel ont- wikkeltijd besparen als een klasse die voor een bepaalde toepassing gemaakt is, ook in een andere toepassing kan worden ingezet. Het belang van deze eigenschap is u in feite al bekend: de Java API is in wezen niets anders dan een verzameling herbruikbare klassen
